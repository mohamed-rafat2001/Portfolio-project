1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd Portfolio-project
   ```

2. **Setup Server**:
   ```bash
   cd server
   npm install
   # Create a .env file based on the environment variables needed
   npm run dev
   ```

3. **Setup Client**:
   ```bash
   cd client
   npm install
   npm run dev
   ```
